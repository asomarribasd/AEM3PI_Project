COURSE CATALOG AND COURSE DESCRIPTIONS

PYTHON FUNDAMENTALS COURSE
Course Code: PY101
Duration: 8 weeks, 24 hours
Level: Beginner
Prerequisites: None
Description: Learn Python programming from scratch. Covers variables, data types, control flow, functions, and modules. No prior programming experience required. Includes hands-on coding exercises and projects. Certificate awarded upon completion.

WEB DEVELOPMENT WITH HTML, CSS, AND JAVASCRIPT
Course Code: WEB201
Duration: 10 weeks, 30 hours
Level: Beginner to Intermediate
Prerequisites: Computer basics
Description: Create modern, responsive websites. Learn HTML5 for structure, CSS3 for styling, and JavaScript for interactivity. Build real projects including a portfolio website. Includes career guidance for web developers.

DATA SCIENCE FUNDAMENTALS
Course Code: DS301
Duration: 12 weeks, 36 hours
Level: Intermediate
Prerequisites: Python Fundamentals
Description: Master data analysis using Python, pandas, and matplotlib. Learn statistical concepts and data visualization. Real-world datasets used in projects. Career path guidance for data scientists included.

MACHINE LEARNING ESSENTIALS
Course Code: ML401
Duration: 14 weeks, 42 hours
Level: Intermediate to Advanced
Prerequisites: Python Fundamentals, Data Science Fundamentals
Description: Introduction to machine learning algorithms including supervised and unsupervised learning. Hands-on projects using scikit-learn and TensorFlow. Build predictive models. Industry-ready skills taught.

ADVANCED PYTHON FOR DATA ENGINEERING
Course Code: PY402
Duration: 10 weeks, 30 hours
Level: Advanced
Prerequisites: Python Fundamentals
Description: Learn software engineering best practices in Python. Includes design patterns, testing, debugging, and performance optimization. Real-world data engineering problems solved. Expert-level guidance.

CLOUD COMPUTING FUNDAMENTALS
Course Code: CLOUD201
Duration: 8 weeks, 24 hours
Level: Intermediate
Prerequisites: Linux basics recommended
Description: Introduction to AWS, Azure, and Google Cloud. Learn cloud infrastructure, storage, and computing. Hands-on labs in each platform. Cloud certification prep included.

MOBILE APP DEVELOPMENT WITH REACT NATIVE
Course Code: MOBILE301
Duration: 12 weeks, 36 hours
Level: Intermediate
Prerequisites: JavaScript fundamentals
Description: Build native mobile apps for iOS and Android using React Native. Learn state management, navigation, and APIs. Deploy apps to app stores. Real app development skills.

DATABASE DESIGN AND SQL MASTERY

Level: Intermediate
Prerequisites: Basic programming knowledge

CYBERSECURITY FUNDAMENTALS
